# g-o_localisation
La géolocalisation permet de localiser un utilisateur ou un objet sur une carte et de calculer l’itinéraire optimal entre deux points. Ce projet utilise des données GPS et une API de cartographie pour afficher la distance, la durée, et tracer la meilleure route (verte), la plus longue (rouge), des alternatives (bleu) et une ligne directe (noir)
